# OmegaKit MiniSumo

Repositório dedicado aos projetos de placas de circuito impresso (PCI) desenvolvidas para a categoria Mini Sumô - 500g (Autônomo | RC).

## OmegaKit FX50

Esta pasta diz respeito à versão da placa desenhada para os sensores FX50, da Fox Dynamics. A principal diferença deste desenho para o OmegaKit_RZ60 é a presença de LEDs para a sinalização de recepção de sinal.

[Datasheet FX50](https://github.com/luisf18/FXDevices/blob/main/Sensor_FXS50/Datasheet_FX_S50.pdf)

[Projeto esquemático OmegaKit FX50](./OmegaKit_FX50/OmegaKit_FX50.pdf)

## OmegaKit RZ60

Esta pasta diz respeito à versão da placa desenhada para os sensores RZ60s, da Sumozade. Esta placa liga diretamente os sensores à placa, visto que há LEDs na parte de trás dos sensores para sinalização de detecção.

[Informações RZ60s](https://sumozade.com/rz60s-digital-infrared-sensor--0-60-cm-)

[Projeto esquemático OmegaKit RZ60](./OmegaKit_RZ60/OmegaKit_RZ60.pdf)

## Omegakit Junior

Esta pasta diz respeito à versão da placa dedicada à nova categoria de mini sumô autônomo para juniores. Ela utiliza um Arduino Nano ao invés de um ESP32 DEVKIT V1, e possui 5 entradas analógicas para uso, visto que o Arduino Nano pode converter entradas analógicas em digitais de acordo com a necessidade.

[Projeto esquemático OmegaKit Junior](./OmegaKit_Junior/OmegaKit_Junior.pdf)
